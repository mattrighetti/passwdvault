## Getting started
### Installing
#### Installing Badger Command Line Tool Manually
```sh
$ git clone https://github.com/MattRighetti/passwdvault.git
$ cd passwdvault
$ go build
$ ln -s $(pwd)/passwdvault /usr/local/bin/passwdvault
```
